Algorithm: Calculate Relative Error

Start

Step 1:
    Read the true value (T).

Step 2:
    Read the approximate value (A).

Step 3:
    Check whether the true value is zero.
        If T = 0:
            Display "Relative error is undefined because the true value is zero."
            Stop.

Step 4:
    Calculate the absolute error.
        Absolute Error = |T − A|

Step 5:
    Calculate the relative error.
        Relative Error = Absolute Error / |T|

Step 6:
    Display the relative error.

End
