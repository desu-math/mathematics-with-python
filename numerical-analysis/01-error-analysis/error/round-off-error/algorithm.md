# Algorithm: Calculating Round-off Error

**Input:**

* An exact value.
* The number of decimal places to which the value should be rounded.

**Output:**

* The rounded value.
* The round-off error.

---

## Algorithm

**Step 1**

Read the exact value.

**Step 2**

Read the required number of decimal places.

**Step 3**

Round the exact value to the specified number of decimal places.

**Step 4**

Calculate the round-off error.

round-off error = exact value - rounded value

**Step 5**

Display

* the exact value,
* the rounded value,
* and the round-off error.

**End**
