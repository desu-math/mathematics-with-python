Algorithm: Calculate Percentage Error

Start

Step 1:
    Read the true value (T).

Step 2:
    Read the approximate value (A).

Step 3:
    If T = 0,
        Display "Percentage error is undefined because the true value is zero."
        Stop.

Step 4:
    Calculate the absolute error.
        Absolute Error = |T − A|

Step 5:
    Calculate the relative error.
        Relative Error = Absolute Error / |T|

Step 6:
    Calculate the percentage error.
        Percentage Error = Relative Error × 100

Step 7:
    Display the percentage error.

End
