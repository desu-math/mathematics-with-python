# Types of Vectors

## Introduction

Vectors can be classified according to their magnitude, direction, or position. Understanding the different types of vectors helps describe and analyze vector quantities in mathematics, physics, engineering, and computer science.

---

# Zero Vector

A **zero vector** is a vector whose magnitude is zero.

It is denoted by

```text
0
```

or

```text
(0, 0)
```

in two dimensions, and

```text
(0, 0, 0)
```

in three dimensions.

A zero vector has **no direction** because its magnitude is zero.

---

# Unit Vector

A **unit vector** is a vector whose magnitude is exactly **1**.

A unit vector indicates direction only.

For a vector

```text
v = (x, y)
```

the corresponding unit vector is

```text
u = v / |v|
```

Common unit vectors are

```text
i = (1, 0)
j = (0, 1)
k = (0, 0, 1)
```

which point along the x-, y-, and z-axes, respectively.

---

# Equal Vectors

Two vectors are **equal** if they have:

* the same magnitude, and
* the same direction.

Their positions do not matter.

Example:

```text
(3, 4) and (3, 4)
```

are equal vectors.

---

# Negative Vectors

The **negative** of a vector has the same magnitude but the opposite direction.

If

```text
v = (x, y)
```

then

```text
-v = (-x, -y)
```

Example:

```text
v = (2, 5)

-v = (-2, -5)
```

---

# Parallel Vectors

Two vectors are **parallel** if they have the same or opposite direction.

One vector is a scalar multiple of the other.

Example:

```text
(2, 4)

(1, 2)
```

are parallel because

```text
(2, 4) = 2(1, 2)
```

---

# Anti-parallel Vectors

Two vectors are **anti-parallel** if they are parallel but point in opposite directions.

Example:

```text
(2, 4)

(-1, -2)
```

---

# Position Vector

A **position vector** represents the location of a point relative to the origin.

If

```text
P(x, y)
```

is a point, then its position vector is

```text
OP = (x, y)
```

where **O** is the origin.

---

# Summary

* A **zero vector** has zero magnitude.
* A **unit vector** has magnitude one.
* **Equal vectors** have the same magnitude and direction.
* **Negative vectors** have the same magnitude but opposite direction.
* **Parallel vectors** are scalar multiples of one another.
* **Anti-parallel vectors** are parallel vectors with opposite directions.
* A **position vector** locates a point relative to the origin.
